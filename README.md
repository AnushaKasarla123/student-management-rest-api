# Student Management REST API

A CRUD REST API built using Spring Boot, Spring Data JPA, MySQL, Maven, and Postman.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

## Features
- Add Student
- View All Students
- View Student by ID
- Update Student
- Delete Student

## API Endpoints

| Method | Endpoint |
|--------|-----------|
| POST | /students |
| GET | /students |
| GET | /students/{id} |
| PUT | /students/{id} |
| DELETE | /students/{id} |

## Database

MySQL

Table: students
